## Description

A powerful WooCommerce plugin that replaces dull dropdowns with vibrant color swatches, stylish buttons, and image-based options making product selection clearer, faster, and more visually engaging for your customers.

- [Read Official Documentation](https://www.gethandytools.site/plugin/handy-variation-swatches-for-woocommerce/)
- [Try Live Demo Shop](https://www.handytoolsdemo.shop)

## Why You Should Use It?

Explore the comprehensive list of features and compelling reasons to choose Handy Variation Swatches for your WooCommerce store.

- **Super Easy To Use** - The Customizer API can be a headache and confusing. So Handy managed to simplify the process of adding controls to customizer.
- **Beautiful & Engaging Swatches** - Replace dropdowns with stylish color, image, and button swatches that auto-switch product images and beautifully match any theme.
- **Supports Color Swatches** - Display product attributes using vibrant color blocks. Perfect for clothing, bags, or any item with color variations, giving users a clearer visual representation.
- **Supports Image Swatches** - Use images to showcase variations like patterns, materials, or styles. Customers can quickly identify product differences and choose the exact variation they want.
- **Supports Label/Button Swatches** - Convert variations into clean, readable button labels. Best for sizes, dimensions, capacity, materials, or any attribute that works better with text rather than colors or images.
- **Supports Multi-Color Swatches** - Enable swatches that display two or more colors within a single variation block perfect for products that come in mixed tones, gradients, multicolor patterns, or dual-shade designs.
- **Tooltips Text, HTML & Image** - Enhance user clarity by showing text descriptions, HTML markup or thumbnail images inside tooltips. This is especially helpful when variations have subtle differences.
- **Customizable Swatch Style** - Customize swatches with flexible shapes, sizes, spacing, and effects, plus styled tooltips and full layout control for pixel-perfect design.
- **Global & Per-Product Setup** - Control swatch settings globally for your entire store or override them at the product level for custom layouts. This gives store owners maximum flexibility and fine tuning.
- **Seamless Theme Compatibility** - Works smoothly with most modern WooCommerce themes. The plugin integrates without breaking layout structure or requiring heavy modifications.

## Installation

### Manual Installation via FTP
- Download the plugin package to your computer.
- Extract the downloaded file (if it is in .zip format).
- Using your preferred FTP client, upload the extracted plugin folder to the `/wp-content/plugins/` directory of your WordPress installation.
- Log in to your WordPress dashboard.
- Go to Plugins → Installed Plugins and click Activate next to the plugin name.

### Manual Installation via WordPress Admin (Upload Method)
- Download the plugin archive (.zip file).
- Log in to your WordPress dashboard.
- Navigate to Plugins → Add New.
- Click the Upload Plugin button at the top of the page.
- Select the downloaded .zip file and click Install Now.
- After installation completes, click Activate to enable the plugin.
